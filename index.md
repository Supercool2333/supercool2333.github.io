## üîß Running Locally

If you want to run this application on your local machine, follow these additional steps:

1. **Install Node.js:** Make sure you have Node.js installed. You can download it from the [official website](https://nodejs.org/).

2. **Clone the Repository:** Open your terminal and run the following command:
   ```bash
   git clone https://github.com/Supercool2333/mern-notes.git
   ```
3. **Navigate to the Directory:** Change into the cloned repository directory.
   ```bash
   cd mern-notes
   ```

4. **Install Dependencies:** Run the following command to install the necessary packages.
   ```bash
   npm install
   ```

5. **Start the Application:** Use the command below to run the application locally.
   ```bash
   npm start
   ```

6. **Open in Browser:** Once the server is running, navigate to `http://localhost:3000` in your web browser to see the application in action.
